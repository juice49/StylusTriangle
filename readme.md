# Stylus Triangle Generator

A little mixin for generating CSS triangles with Stylus.

## Usage

````
triangle(width, height, direction, color)
````

````
.make-me-a-triangle-baby
	triangle(10px, 10px, 'bottom', #000)
````

## npm
	npm install stylus-triangle
